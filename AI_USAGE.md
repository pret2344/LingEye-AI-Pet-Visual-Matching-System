# AI工具使用说明
本项目开发全过程使用AI工具作为辅助，提升单人开发效率。所有核心需求、架构设计、业务逻辑均由本人独立完成，AI仅提供内容生成、思路参考与效率辅助。

## 使用明细总表
| 开发阶段 | AI工具名称 | 使用场景 | 人工修改比例 |
|----------|------------|----------|--------------|
| 可行性分析 | 豆包 | 文档框架搭建、行业数据整理、技术方案对比 | 约40% |
| 需求分析 | 豆包 | 需求条目梳理、用例流程撰写、数据表设计建议 | 约35% |
| 概要设计 | 豆包 | 分层架构梳理、模块边界划分、ER图设计辅助 | 约40% |
| 详细设计 | 豆包 | 算法原理说明、接口参数整理、类结构定义 | 约35% |
| 编码实现 | 豆包 | 问题排查、Bug定位、代码规范优化建议 | 约30% |
| 编码实现 | TRAE | 前端页面代码生成、UI样式优化、模拟数据填充 | 约20% |
| 测试阶段 | 豆包 | 测试用例扩充、Bug根因分析、报告框架生成 | 约45% |
| 文档撰写 | 豆包 | GB格式规范对齐、专业术语统一、内容结构化整理 | 约25% |

## 分阶段详细说明

### 1. 可行性分析阶段
- **工具**：豆包
- **提示词/对话摘要**：
  1. 依据GB/T 8567标准生成可行性分析报告的完整章节大纲
  2. 整理2025年中国宠物行业的市场规模、走失率等行业数据
  3. 对比微信原生小程序、uni-app、Taro三种前端方案的优劣势
  4. 梳理学生项目的风险点并给出应对措施建议
- **人工修改比例**：约40%。AI生成框架与素材，本人补充项目具体细节、选型决策与数据核实。

### 2. 需求分析阶段
- **工具**：豆包
- **提示词/对话摘要**：
  1. 梳理宠物寻宠平台的功能需求清单，按优先级分级
  2. 生成用户用例的标准描述模板
  3. 根据业务场景设计MySQL数据表字段与约束
  4. 给出小程序项目的非功能性能指标参考
- **人工修改比例**：约35%。AI提供基础清单，本人结合项目实际裁剪功能、定义优先级与业务规则。

### 3. 概要设计阶段
- **工具**：豆包
- **提示词/对话摘要**：
  1. 设计B/S三层架构的分层职责与交互逻辑
  2. 按业务领域划分功能模块，明确模块边界
  3. 梳理数据库实体关系，给出索引设计建议
  4. 制定RESTful接口的统一返回格式与错误码规范
- **人工修改比例**：约40%。AI提供通用方案，本人适配项目规模、技术栈与资源约束。

### 4. 详细设计阶段
- **工具**：豆包
- **提示词/对话摘要**：
  1. 讲解余弦相似度算法的原理，给出数学公式与JS伪代码
  2. 生成登录、发布、匹配核心流程的时序步骤
  3. 定义核心类的属性与方法，包含继承关系
  4. 细化接口的入参出参字段与校验规则
- **人工修改比例**：约35%。AI补充原理细节与格式，本人确定业务逻辑、参数规则与实现方案。

### 5. 编码实现阶段
#### 5.1 豆包
- **提示词/对话摘要**：
  1. 分析小程序图片不显示的可能原因，给出排查步骤
  2. 解释Express全局异常中间件的写法
  3. 提供JWT鉴权中间件的实现代码
  4. 排查接口跨域、数据库连接失败等问题
- **人工修改比例**：约30%。AI提供排错思路与代码片段，本人调试验证、适配项目结构。

#### 5.2 TRAE
- **提示词/对话摘要**：
  1. 基于项目目录生成首页瀑布流列表的WXML、WXSS、JS代码
  2. 生成AI匹配页、发布页、个人中心的页面基础代码
  3. 优化卡片布局、配色、圆角阴影等UI样式
  4. 补充列表模拟数据与状态标签
- **人工修改比例**：约20%。AI生成前端初稿，本人完成接口联调、逻辑修正、功能验证与细节优化。

### 6. 测试阶段
- **工具**：豆包
- **提示词/对话摘要**：
  1. 生成小程序项目的功能测试用例，覆盖正常与异常场景
  2. 补充边界测试与安全测试的测试点
  3. 分析常见前端Bug的根因与修复方向
  4. 按照GB标准生成测试报告的结构框架
- **人工修改比例**：约45%。AI扩充用例场景，本人执行实际测试、记录结果、统计数据。

### 7. 文档撰写阶段
- **工具**：豆包
- **提示词/对话摘要**：
  1. 对照国家标准调整文档格式、术语与编号
  2. 将零散内容整理为结构化的章节内容
  3. 润色专业表述，统一文档文风
- **人工修改比例**：约25%。AI辅助格式与表达，核心内容与结论均由本人撰写。
